#### Abstracts
##### **Act Wisely: Cultivating Meta-Cognitive Tool Use in Agentic Multimodal Models**
2604.08545v1 by Shilin Yan, Jintao Tong, Hongwei Xue, Xiaojun Tang, Yangyang Wang, Kunyu Shi, Guannan Zhang, Ruixuan Li, Yixiong Zou

The advent of agentic multimodal models has empowered systems to actively interact with external environments. However, current agents suffer from a profound meta-cognitive deficit: they struggle to arbitrate between leveraging internal knowledge and querying external utilities. Consequently, they frequently fall prey to blind tool invocation, resorting to reflexive tool execution even when queries are resolvable from the raw visual context. This pathological behavior precipitates severe latency bottlenecks and injects extraneous noise that derails sound reasoning. Existing reinforcement learning protocols attempt to mitigate this via a scalarized reward that penalizes tool usage. Yet, this coupled formulation creates an irreconcilable optimization dilemma: an aggressive penalty suppresses essential tool use, whereas a mild penalty is entirely subsumed by the variance of the accuracy reward during advantage normalization, rendering it impotent against tool overuse. To transcend this bottleneck, we propose HDPO, a framework that reframes tool efficiency from a competing scalar objective to a strictly conditional one. By eschewing reward scalarization, HDPO maintains two orthogonal optimization channels: an accuracy channel that maximizes task correctness, and an efficiency channel that enforces execution economy exclusively within accurate trajectories via conditional advantage estimation. 摘要：在三維空間中進行草圖繪製使得對形狀、結構和空間關係的表達性推理成為可能，但通過自然語言生成三維草圖仍然是一大挑戰。在這項工作中，我們介紹了3DrawAgent，一個無需訓練的、以語言驅動的三維草圖生成框架，利用大型語言模型（LLMs）在幾何反饋下順序繪製三維貝茲曲線。與以往的2D草圖代理不同，我們的方法引入了一種相對經驗優化策略，適應了最近提出的群體獎勵政策優化（GRPO）範式。我們不依賴明確的真實監督，而是構建生成草圖之間的成對比較，每對由一個相對較好和一個較差的結果組成，這基於基於CLIP的感知獎勵和基於LLM的細緻質量評估。這些經驗然後用於迭代地細化三維繪圖的先前知識，使模型的三維意識得到黑箱強化。這一設計使我們的模型能夠在不進行參數更新的情況下自我提升其空間理解和繪圖質量。實驗表明，3DrawAgent可以從多樣的文本提示中生成複雜且連貫的三維貝茲草圖，展現出新興的幾何推理，並能夠對新形狀進行泛化，為推進無需訓練的三維草圖智能領域建立了一個新的範式。
